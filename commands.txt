venv\Scripts\activate - запуск виртуальной среды
pip install <package name> - установить пакет в проект
pip list - список пакетов в проекте
flask run - запуск приложения фласк
flask db init -инициализация базы данных
flask db migrate -m "сообщение о внесённых изменениях" - миграция в базе данных, нужна при внесении изменений в структуру
flask db upgrade / downgrade - обновить базу данных / откатить до предыдущей версии
user = User(username='имя', email='почта') - создание нового пользователя
db.session.add(user) - добавить новое поле в базу данных
db.session.commit() - сохранить изменения в базе данных
users = User.query.all() - получение всех пользователей из базы данных
user = User.query.get("здесь id") - получение одного конкретного пользователя из базы
flask shell - оболочка приложения
